1. Array Rotation (Left or Right by a specified number of positions)

Question: Write a Java program to rotate an array to the left or right by a specified number of positions. 
import java.util.Scanner;

public class ArrayRotation {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Input: Array, number of positions, and direction
        System.out.print("Enter the number of positions to rotate: ");
        int positions = sc.nextInt();
        System.out.print("Enter the direction (Left/Right): ");
        String direction = sc.next();

        int[] arr = {1, 2, 3, 4, 5, 6, 7};

        System.out.println("Original Array: ");
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();

        // Perform rotation
        if (direction.equalsIgnoreCase("Left")) {
            rotateLeft(arr, positions);
        } else if (direction.equalsIgnoreCase("Right")) {
            rotateRight(arr, positions);
        } else {
            System.out.println("Invalid direction input.");
        }

        System.out.println("Rotated Array: ");
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }

    // Rotate array to the left by 'positions' places
    private static void rotateLeft(int[] arr, int positions) {
        int len = arr.length;
        int[] temp = new int[len];
        for (int i = 0; i < len; i++) {
            temp[i] = arr[(i + positions) % len];
        }
        System.arraycopy(temp, 0, arr, 0, len);
    }

    // Rotate array to the right by 'positions' places
    private static void rotateRight(int[] arr, int positions) {
        int len = arr.length;
        int[] temp = new int[len];
        for (int i = 0; i < len; i++) {
            temp[i] = arr[(i - positions + len) % len];
        }
        System.arraycopy(temp, 0, arr, 0, len);
    }
}
Sample Input:
Array: {1, 2, 3, 4, 5, 6, 7}
Number of positions: 2
Direction: Left
Sample Output:
Rotated Array: [3, 4, 5, 6, 7, 1, 2]

2. Find the Missing Number in a Sequence

Question: Write a Java program to find the missing number in a sequence of integers from 1 to N. 
public class MissingNumber {
    public static void main(String[] args) {
        int[] arr = {1, 2, 4, 5, 6};
        int n = 6; // Range from 1 to 6

        int expectedSum = (n * (n + 1)) / 2;
        int actualSum = 0;

        for (int num : arr) {
            actualSum += num;
        }

        int missingNumber = expectedSum - actualSum;
        System.out.println("Missing Number: " + missingNumber);
    }
}
Sample Input:
Array: {1, 2, 4, 5, 6}
Range: 1 to 6
Sample Output:
Missing Number: 3

3. Find All Pairs in an Array Whose Sum Equals a Given Target

Question: Write a Java program to find all pairs in an array whose sum equals a given target. 
import java.util.HashSet;

public class PairSum {
    public static void main(String[] args) {
        int[] arr = {2, 4, 3, 5, 7, 8, 1};
        int target = 9;

        System.out.println("Pairs with sum " + target + ":");
        HashSet set = new HashSet<>();

        for (int num : arr) {
            int complement = target - num;
            if (set.contains(complement)) {
                System.out.println("(" + num + ", " + complement + ")");
            }
            set.add(num);
        }
    }
}
Sample Input:
Array: {2, 4, 3, 5, 7, 8, 1}
Target Sum: 9
Sample Output:
Pairs with sum 9:
(4, 5)
(2, 7)
(8, 1)

4. Traffic Light System Using Switch Statement

Question: Write a Java program to simulate a traffic light system using a switch statement. The program should display messages based on the color (Red, Yellow, Green). 
import java.util.Scanner;

public class TrafficLight {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // Input: Traffic Light Color
        System.out.print("Enter the traffic light color (Red/Yellow/Green): ");
        String color = sc.nextLine();

        switch (color.toLowerCase()) {
            case "red":
                System.out.println("STOP! Wait until the light turns green.");
                break;
            case "yellow":
                System.out.println("GET READY! The light is about to turn green.");
                break;
            case "green":
                System.out.println("GO! You can proceed.");
                break;
            default:
                System.out.println("Invalid color.");
        }
    }
}
Sample Input:
Traffic Light Color: Red
Sample Output:
STOP! Wait until the light turns green.

5. Income Tax Calculation Based on Salary

Question: Write a Java program to calculate the income tax of an employee based on the following slabs:
- Income <= 2,50,000: No tax
- Income <= 5,00,000: 5%
- Income <= 10,00,000: 20%
- Income > 10,00,000: 30%
import java.util.Scanner;

public class IncomeTax {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Input: Salary
        System.out.print("Enter your salary: ");
        double salary = sc.nextDouble();

        double tax = 0;

        if (salary <= 250000) {
            tax = 0;
        } else if (salary <= 500000) {
            tax = (salary - 250000) * 0.05;
        } else if (salary <= 1000000) {
            tax = (250000 * 0.05) + (salary - 500000) * 0.20;
        } else {
            tax = (250000 * 0.05) + (500000 * 0.20) + (salary - 1000000) * 0.30;
        }

        System.out.println("Income Tax to be paid: ₹" + tax);
    }
}
Sample Input:
Salary: 8,50,000
Sample Output:
Income Tax to be paid: ₹85,000
