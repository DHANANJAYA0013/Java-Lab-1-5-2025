1. Circle Class with Radius Attribute

Question: Write a Java program to create a class called "Circle" with a radius attribute. You can access and modify this attribute. Calculate the area and circumference of the circle. 
class Circle {
    private double radius;

    public Circle(double radius) {
        this.radius = radius;
    }

    public double getRadius() {
        return radius;
    }

    public void setRadius(double radius) {
        this.radius = radius;
    }

    public double calculateArea() {
        return Math.PI * radius * radius;
    }

    public double calculateCircumference() {
        return 2 * Math.PI * radius;
    }

    public static void main(String[] args) {
        Circle circle = new Circle(5.0);
        System.out.println("Radius: " + circle.getRadius());
        System.out.println("Area: " + circle.calculateArea());
        System.out.println("Circumference: " + circle.calculateCircumference());
    }
}
Sample Input:
Radius = 5
Sample Output:
Radius: 5.0
Area: 78.53981633974483
Circumference: 31.41592653589793

2. TrafficLight Class with Color and Duration

Question: Write a Java program to create a class called "TrafficLight" with attributes for color and duration, and methods to change the color and check for red or green. 
class TrafficLight {
    private String color;
    private int duration;

    public TrafficLight(String color, int duration) {
        this.color = color;
        this.duration = duration;
    }

    public void changeColor(String newColor) {
        this.color = newColor;
    }

    public boolean isRed() {
        return this.color.equalsIgnoreCase("Red");
    }

    public boolean isGreen() {
        return this.color.equalsIgnoreCase("Green");
    }

    public void displayTrafficLight() {
        System.out.println("Current Color: " + color);
        System.out.println("Duration: " + duration + " seconds");
    }

    public static void main(String[] args) {
        TrafficLight trafficLight = new TrafficLight("Red", 60);
        trafficLight.displayTrafficLight();
        trafficLight.changeColor("Green");
        trafficLight.displayTrafficLight();
        System.out.println("Is it red? " + trafficLight.isRed());
        System.out.println("Is it green? " + trafficLight.isGreen());
    }
}
Sample Input:
Color = Red, Duration = 60
Sample Output:
Current Color: Red
Duration: 60 seconds
Current Color: Green
Duration: 60 seconds
Is it red? false
Is it green? true

4. Vehicle Class with Constructor Overloading

Question: Create a Java class "Vehicle" to store vehicle details. Implement constructor overloading to register vehicles with different levels of information: - A constructor that accepts only vehicle model. - A constructor that accepts vehicle model and manufacturer name. - A constructor that accepts vehicle model, manufacturer name, and registration number. 
class Vehicle {

    private String manufacturer;
    private String registrationNumber;

    public Vehicle(String model) {
        this.model = model;
    }

    public Vehicle(String model, String manufacturer) {
        this.model = model;
        this.manufacturer = manufacturer;
    }

    public Vehicle(String model, String manufacturer, String registrationNumber) {
        this.model = model;
        this.manufacturer = manufacturer;
        this.registrationNumber = registrationNumber;
    }

    public void displayDetails() {
        System.out.println("Model: " + model);
        System.out.println("Manufacturer: " + manufacturer);

    public static void main(String[] args) {
        Vehicle vehicle1 = new Vehicle("Model X");

        Vehicle vehicle3 = new Vehicle("Model Z", "Manufacturer B", "123XYZ");

        vehicle1.displayDetails();
        vehicle2.displayDetails();
        vehicle3.displayDetails();
    }
}

5. Library Book Management System

Question: Develop a Java program for a Library Book Management System where users can add book details. Create a "Book" class with attributes like title, author, and price. Implement a mechanism to take user input using either a constructor or a separate method to initialize book details. 
import java.util.Scanner;

class Book {
    private String title;
    private String author;
    private double price;

    public Book(String title, String author, double price) {
        this.title = title;
        this.author = author;
        this.price = price;
    }

    public void displayBookDetails() {
        System.out.println("Title: " + title);
        System.out.println("Author: " + author);
        System.out.println("Price: ₹" + price);
    }

    public void getBookDetails() {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter book title: ");
        this.title = sc.nextLine();
        System.out.print("Enter book author: ");
        this.author = sc.nextLine();
        System.out.print("Enter book price: ");
        this.price = sc.nextDouble();
    }

    public static void main(String[] args) {
        Book book1 = new Book("Java Programming", "John Doe", 500);
        book1.displayBookDetails();

        Book book2 = new Book("", "", 0);
        book2.getBookDetails();
        book2.displayBookDetails();
    }
}
Sample Input:
Book Title = "Java Programming"
Book Author = "John Doe"
Price = 500
Sample Output:
Title: Java Programming
Author: John Doe
Price: ₹500.0
Title: (User input)
Author: (User input)
Price: (User input)

6. Entrance Exam Registration System

Question: Write a Java program for an Entrance Exam Registration System that allows students to register with varying details. 
class Candidate {
    private String name;
    private String applicationNumber;
    private String examCenter;

    public void registerCandidate(String name, String applicationNumber) {
        this.name = name;
        this.applicationNumber = applicationNumber;
        System.out.println("Candidate Registered: " + name + " | Application Number: " + applicationNumber);
    }

    public void registerCandidate(String name, String applicationNumber, String examCenter) {
        this.name = name;
        this.applicationNumber = applicationNumber;
        this.examCenter = examCenter;
        System.out.println("Candidate Registered: " + name + " | Application Number: " + applicationNumber + " | Exam Center: " + examCenter);
    }

    public void displayDetails() {
        System.out.println("Name: " + name);
        System.out.println("Application Number: " + applicationNumber);
        System.out.println("Exam Center: " + (examCenter != null ? examCenter : "Not Provided"));
    }

    public static void main(String[] args) {
        Candidate candidate1 = new Candidate();
        candidate1.registerCandidate("Alice", "A123");
        candidate1.displayDetails();

        Candidate candidate2 = new Candidate();
        candidate2.registerCandidate("Bob", "B456", "New York");
        candidate2.displayDetails();
    }
}
Sample Output:
Candidate Registered: Alice | Application Number: A123
Candidate Registered: Bob | Application Number: B456 | Exam Center: New York
